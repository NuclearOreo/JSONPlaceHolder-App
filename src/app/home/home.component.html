Overview

Making Angular App using JSON PlaceHolder as the API for this App. The purpose of this app, is for me to pratice producing an Angular app and then use it reference for myself for future Anuglar projects. This project is going to be a "Single-page application". Showing basic routing between compoments. There will four main compoments displayed on the navbar: Posts, Comments, Photos and Users. The "Posts" compoment will show a list of posts. Each post can be be clicked on to show the contents the post. Same thing goes for the "Comments" compoment. For the "Photos" compoment, it will show a gallery of photos from the API. The last compoment "Users" will display a list of users. Each user can be clicked on to display a profile.
